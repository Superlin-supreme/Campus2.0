一、安装

1.数据库安装：在PHPmyadmin里新建数据库名为test，再导入sql文件。

2.前端ajax调用接口IP更改：在index.js和login.js里有个IP的变量，需修改成当前服务器的ip地址或者localhost加端口号。

3.campus目录放在服务器网站根目录下。


二、使用

1.以游客身份访问：在登录界面点击“以游客身份进入”可进入主界面，可以浏览单子，但不可进行其他操作。

2.注册账号：在登录界面点击“注册”可以填写信息注册账号，需填写正确的手机号码和邮箱地址。

3.登录账号：在登录界面输入手机号或邮箱和密码，点击“登录”，验证成功后会进入首页界面，并保留登录状态1小时。

4.若登录状态下再次访问登录界面会出现已登录者信息，不会弹出登录和注册窗口。

5.发布和管理个人订单：在首页点击“发布订单”，正确填写订单信息后可点击“发布”进行发布，发布成功后可在首页看到自己的订单，可进行删除。同时可以点击“我的发布”进入我的全部订单界面，若有人接单，会显示接单人的信息（名称，手机号，邮箱）。

6.评论和回复：在浏览订单时，可以点击“n条疑问”的按钮查看其它用户对订单的疑问，可以输入自己的疑问进行评论，也可以点击“回复”输入对该用户的回复。

7.查看其它用户的主页：在浏览订单时，点击其它用户的头像或名称进入其他用户的主页界面，可以看到其他用户的部分信息和该用户的订单，亦可以点击“申请好友”申请添加好友。

8.在线聊天界面：点击右上角的消息按钮或在好友的主页点击私信，可以进入在线聊天界面，在好友列表里点击选择好友即可开始聊天，在聊天框输入消息并点击“发送”可发送聊天信息，若好友发来了消息，会立即显示在消息窗口！

9.添加和删除好友：在好友申请和管理框里，可删除已经添加的好友，可看到申请加好友的信息，可接受或拒绝。在输入框中输入需添加好友的手机号可以申请添加好友。

10.个人信息界面：点击右上角的头像再点击我的主页可进入我的信息管理界面，点击任一条信息后的修改按钮可进行修改操作，点击“修改密码”可进行修改密码的操作。
